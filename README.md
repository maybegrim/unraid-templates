# Unraid Docker Templates

Community Application templates for [Unraid](https://unraid.net/).

## Applications

| Application | Description |
|---|---|
| [NextExplorer](https://github.com/nxzai/NextExplorer) | Modern, self-hosted web-based file explorer with secure access control |

## Installation

These templates are available through the Unraid Community Applications plugin (once approved). Search for **NextExplorer** in the Apps tab.

### Manual Installation

1. In the Unraid web UI, go to **Docker → Add Container → Template Repositories**
2. Add this URL: `https://github.com/maybegrim/unraid-templates`
3. Click **Save**, then search for the desired application
